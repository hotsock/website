# Overview

Interactive examples showcasing Hotsock's real-time capabilities. Each example includes a live demo and a walkthrough of how it works.

- <Link to="real-time-chat">Real-Time Chat</Link> — Two users exchange messages
  in real time with typing indicators.
- <Link to="live-dashboard">Live Dashboard</Link> — A metrics dashboard
  receiving real-time server-published updates.
- <Link to="collaborative-todo">Collaborative To-Do List</Link> — Two users
  manage a shared to-do list with message storage and history.
- <Link to="notification-feed">Notification Feed</Link> — A notification bell
  with real-time delivery using channel aliases and auto-subscribe.
